== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woorpd` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->WooCommerce Remote Products Display screen to configure the plugin.
